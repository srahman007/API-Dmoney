# API-Dmoney

## Project summary:
This project has automate user API from postman using newman

## Technology used:
- Postman
- Newman

## Pre-requisites:
- NodeJS

## How to run?
- ``` git clone ```
- ``` npm i ```
- ``` npm test ```
A report will be generate in your project source root
